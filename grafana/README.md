# Home Assistant Add-on: Grafana

The open platform for beautiful analytics and monitoring.

## About

Grafana allows you to query, visualize, alert on and understand your metrics
no matter where they are stored. Create, explore, and share dashboards.

This add-on is a downstream port of the Home Assistant Community Add-on
[`hassio-addons/addon-grafana`](https://github.com/hassio-addons/addon-grafana)
packaging Grafana 13.x, maintained under `smalluniverse-net` so new Grafana
releases can be picked up as soon as they land upstream instead of waiting
for community addon cuts.

See `DOCS.md` for configuration and usage instructions (schema, ingress,
plugin install, SSL, env vars).
